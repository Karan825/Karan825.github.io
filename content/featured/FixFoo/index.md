Engineered an autonomous GitHub Issue-to-PR AI agent using FastAPI, GitHub Actions, E2B Cloud, and Google Gemini SDK that analyzes issues, generates code fixes, validates them in isolated sandboxes, and opens verified PRs. Implemented a production-grade ReAct execution pipeline with automatic repository language detection (Python, Node.js, Go, Rust), end-to-end test execution, and coverage-delta validation to ensure only verified fixes are proposed. Designed a secure autonomous execution pipeline with sandbox isolation, protected filesystem policies, output sanitization, and verification gates to prevent unsafe code changes.
